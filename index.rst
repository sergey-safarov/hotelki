Хотелки высказанные клиентами
=============================

Веб доступ к записям разговоров
--------------------------------

В настоящее время записи телефонных переговоров хранятся на сервере "Русцентросвязь" и недоступны пользователям. Для реализации веб доступа
необходимо осуществить доработку модуля CDR платформы Kazoo по следущим пунктам:

1) генерация ссылки на извлечение аудиозаписи из базы;
2) аутентификация и авторизация доступа к аудиозаписям;
3) задание срока хранения записей с автоматическим удалением старых и внесением данных о дате удаления записи (при возможности);
4) API для получения среднего размера архива записей за месяц;
5) добавление элементов управления проигрыванием записей на журнале CDR и функции удаления выбранных записей;


Расширенная аналитика по звонкам
--------------------------------

Требуется формировать следующие отчеты:

1) количество звонков пропущенных из-за занятости телефонов;
2) количество звонков оставленных без ответа (абонента "А" не дозвонился)
3) количество звонков когда абоненту ответил робот и его звонок поставлен в очередь;
4) количество звонков когда абонент находился в очереди и не дождался ответа оператора;
5) данные о количестве принятых звонков операторами (сотрудниками) в разрезе по сотрудникам;
6) данные о среднем времени ответа сотрудниками на звонки в разрезе по сотрудникам;
7) данные о средней длительности звонков сотрудников в разрезе по сотрудникам;
8) данные о количестве не отвеченных звонков сотрудниками в разрезе по сотрудникам.
